# RPG 

    make && ./RPG

A minimalisic RPG with : 

* tilemap
* dialog
* music

# Level Editor

Create or modify a .map file. 

## build 

    make -f Makefile_level_editor

## run 

-l level_path

    ./level_editor -l data/overworld.map
